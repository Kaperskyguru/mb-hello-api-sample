# hello-api-sample

Your Node.js backend starter. It's already runnable.

## Run
Click **Run Server** (or in the terminal: `npm install && npm run dev`).
The server listens on the port the playground expects (`process.env.PORT`),
so leave `const port = process.env.PORT || 8080` as-is.

## Endpoints
- `GET /` → a hello payload
- `GET /health` → health check

Edit `index.js` to build out the project's tasks.
